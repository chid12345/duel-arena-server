Что меняется в боевом экране:
1. Над боссом — слой 16 частиц разного размера, плывут вверх (5–7 сек цикл)
2. Туман — мягкий радиальный градиент, дрейфует влево-вправо (9 сек)
3. Цвет подстраивается под boss_glow каждого типа автоматически
4. Срез «земли» — нижние 30% изображения босса плавно становятся прозрачными через CSS mask-image — больше нет нелепой подставки внизу.

Производительность: только CSS-анимации (никакого JS-цикла), GPU-ускорение через transform/opacity. Нагрузка минимальная — даже на слабых телефонах не будет лагов.

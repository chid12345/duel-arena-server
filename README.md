# 🥊 ZenDuelArena Bot

Портативный Telegram бот для быстрых PvP боев в стиле "Камень-Ножницы-Бумага".

## Документация для разработки и ИИ

- Регламент проекта: [`CLAUDE.md`](CLAUDE.md)
- Навигатор по темам (бой, магазин, квесты, соц…): [`docs/INDEX.md`](docs/INDEX.md)

## 🎮 Особенности

- ⚡ **Быстрые бои** - 5 секунд на ход, бой до победы
- 🤖 **Умные боты** - 50 ботов разных уровней с реалистичным ИИ
- 📊 **Прокачка** - 100 уровней, система улучшений
- 💰 **Экономика** - золото и алмазы, магазин
- 🏆 **Рейтинги** - топ игроков, соревнования
- 🎯 **Сбалансированная механика** - криты, уклонения, блоки

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка токена
```bash
# Linux/Mac
export TELEGRAM_BOT_TOKEN='your_bot_token_here'

# Windows (PowerShell)
$env:TELEGRAM_BOT_TOKEN='your_bot_token_here'
```
Шаблон переменных окружения: `.env.example`

### 3. Запуск бота
```bash
python main.py
```

## 🎯 Игровая механика

### ⚔️ Боевая система
- **Раунды:** до победы (пока HP не закончится)
- **Время на ход:** 5 секунд
- **AFK:** 3 пропуска = поражение
- **Зоны атаки:** ГОЛОВА, ТУЛОВИЩЕ, НОГИ
- **Урон:** Сила - (Выносливость/2)
- **Криты:** 15% шанс х2 урона
- **Уклонение:** 5% + улучшения
- **Блоки:** полное совпадение = 0 урона

### 📊 Прокачка
- **Уровни:** 1-100
- **Очки за уровень:** +5 свободных + 1 выносливость
- **Распределение:** Сила, Выносливость, HP
- **Награды за уровень:** золото + опыт

### 💰 Экономика
- **Золото:** основная валюта
- **Алмазы:** премиум валюта
- **Награды:** победа +20 золота, поражение +5 золота
- **Ежедневный бонус:** +20 золота
- **Магазин:** оружие, броня, расходники

## 🤖 Система ботов

### Типы ботов:
- **Новички (1-10 lvl):** базовые характеристики
- **Опытные (11-30 lvl):** средняя сложность
- **Мастера (31-50 lvl):** высокая сложность
- **Легенды (51-80 lvl):** максимальная сложность

### ИИ ботов:
- **Адаптивная тактика:** анализ стиля игрока
- **Поведенческие паттерны:** агрессивный, защитный, сбалансированный
- **Реалистичность:** случайные ошибки, разное время реакции
- **Скрытие:** выглядят как обычные игроки

## 🐛 Устранение проблем

### Ошибка "Токен бота не установлен":
```bash
# Linux/Mac - проверьте переменную окружения
echo $TELEGRAM_BOT_TOKEN

# Linux/Mac - установите токен заново
export TELEGRAM_BOT_TOKEN='your_token'

# Windows (PowerShell) - проверьте переменную окружения
echo $env:TELEGRAM_BOT_TOKEN

# Windows (PowerShell) - установите токен заново
$env:TELEGRAM_BOT_TOKEN='your_token'
```

### Ошибка импорта модулей:
```bash
# Установите все зависимости
pip install -r requirements.txt

# Проверьте версию Python (3.7+)
python --version
```

### Проблемы с базой данных:
```bash
# Linux/Mac - удалите файл БД для пересоздания
rm duel_arena.db

# Windows (PowerShell) - удалите файл БД для пересоздания
Remove-Item duel_arena.db -Force

python main.py  # База создастся автоматически
```

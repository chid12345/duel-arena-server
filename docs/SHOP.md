## SHOP — магазин и предметы

Справка. Читать только когда задача про магазин/ассортимент/покупки.

## Правила добавления предмета (коротко)
- У предмета должны быть: **цена**, **лимит/ограничение** (если даёт силу), **понятный эффект**.
- Если предмет даёт силу (HP/урон/бафф) — обязательно:
  - ограничение по времени/количеству/кулдаун
  - или высокая цена + синки валюты

## Технические правила
- Покупка/списание/выдача — атомарно (одна транзакция, `commit`).
- Не дублировать SQL (общие операции — в `repositories/shop.py` или отдельный модуль магазина).

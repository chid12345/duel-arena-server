# QUESTS — квесты, ежедневки, ивенты

Справка. Читать когда задача про квесты, цепочки, ежедневки, ивенты, условия выдачи наград.

## Границы с другими блоками
- **Награды и валюта** (сколько дать, инфляция) → `ECONOMY.md`
- **Тексты кнопок и сообщений** → `UX_TEXTS.md`
- **Где лежит код в репо** (handlers/repositories) → `ARCHITECTURE.md`

## Правила (коротко)
- У квеста должны быть: **условие**, **награда** (согласовать с экономикой), **где видит игрок** (бот / мини‑приложение).
- Новая цепочка или тип задания → отдельное место в коде по Закону 2 в `CLAUDE.md` (новый дом), не сваливать в «общий хвост».
